## 🎨 Recommandations

### Où Trouver les Images
1. **Unsplash** (https://unsplash.com) - Photos gratuites haute qualité
2. **Pexels** (https://pexels.com) - Banque d'images gratuite
3. **Pixabay** (https://pixabay.com) - Images libres de droits

### Mots-clés de Recherche
- **Technology office** - Pour les bureaux et équipes
- **Web development** - Pour le développement web
- **Mobile app design** - Pour les applications mobiles
- **Cloud computing** - Pour le cloud
- **Cybersecurity** - Pour la sécurité
- **Business team** - Pour l'équipe
- **Professional portrait** - Pour les photos d'équipe
- **Tech project** - Pour les projets

### Spécifications
- **Format** : JPG ou PNG
- **Qualité** : Haute résolution (minimum 1920px de largeur)
- **Poids** : Optimiser pour le web (< 500KB)
- **Style** : Professionnel, moderne, cohérent

## 📝 Note
Les images WhatsApp fournies ont été copiées dans le dossier `images/`.
Vous pouvez les renommer selon vos besoins pour correspondre aux noms listés ci-dessus.

### Exemple de Renommage
```
WhatsApp Image 2026-07-06 at 14.41.57.jpeg → about-preview.jpg
WhatsApp Image 2026-07-06 at 14.41.58.jpeg → project-1.jpg
WhatsApp Image 2026-07-06 at 14.41.59.jpeg → team-meeting.jpg
etc...
```

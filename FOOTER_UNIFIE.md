## 🎨 Responsive Design

Le footer s'adapte automatiquement à tous les écrans :

### Desktop (>768px)
- 4 colonnes côte à côte
- Tous les éléments visibles
- Liens organisés verticalement

### Tablette (768px - 1024px)
- 2 colonnes x 2 rangées
- Réorganisation automatique
- Lisibilité optimale

### Mobile (<768px)
- 1 colonne empilée
- Éléments les uns sous les autres
- Navigation simplifiée

## 🔍 Détails techniques

### Chemins relatifs corrects

**Pour index.html :**
```html
<a href="pages/about.html">À propos</a>
<a href="pages/services.html">Services</a>
```

**Pour les pages dans /pages/ :**
```html
<a href="../index.html">Accueil</a>
<a href="about.html">À propos</a>
```

### Classes CSS utilisées
- `.footer` - Conteneur principal
- `.footer-content` - Grille 4 colonnes
- `.footer-col` - Chaque colonne
- `.footer-logo` - Logo et slogan
- `.footer-title` - Titres de sections
- `.footer-links` - Listes de liens
- `.footer-contact` - Informations de contact
- `.footer-legal` - Liens légaux
- `.footer-bottom` - Copyright
- `.back-to-top` - Bouton retour en haut
- `.whatsapp-float` - Bouton WhatsApp flottant
